| 认证成功路径信任客户端身份标识 | P0 | 🔴 | 认证通过后，直接使用客户端传入的标识（userId/tenantId/role）放行，未使用服务端生成的标识替换 |
| 认证失败降级放行 | P0 | 🔴 | 认证服务不可用/超时/异常时，直接放行而非拒绝访问 |
| 无认证路径可访问受保护资源 | P0 | 🔴 | 任何未显式鉴权的入口（如路径匹配错误、默认放行）可能导致未授权访问 |
| 认证状态与权限校验分离 | P1 | 🟠 | 有身份但无权限校验，或权限校验逻辑与认证逻辑分散在不同层级导致校验遗漏 |
