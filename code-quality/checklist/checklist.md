# 检查结果模板

## 输出格式要求

每项检查输出：
- `✅ 无问题` - 检查通过
- `🔴 P0: 问题描述 - 文件:行号` - 危险问题
- `🟠 P1: 问题描述 - 文件:行号` - 告警问题
- `🟡 P2: 问题描述 - 文件:行号` - 建议问题
- `🔵 P3: 问题描述 - 文件:行号` - 轻微问题

## 问题等级颜色

| 等级 | 颜色 | 说明 |
|------|------|------|
| P0 | 🔴 | 危险，必须立即修复 |
| P1 | 🟠 | 告警，建议修复 |
| P2 | 🟡 | 建议，按计划优化 |
| P3 | 🔵 | 轻微，可选优化 |
